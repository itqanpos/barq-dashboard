# لوحة تحكم برق POS

لوحة تحكم ويب احترافية لتطبيق برق POS - نظام إدارة نقطة البيع الشامل.

## ✨ الميزات

- 📊 **لوحة تحكم شاملة** - عرض الإحصائيات والتقارير الرئيسية
- 💰 **إدارة المبيعات** - تتبع المبيعات والإيرادات
- 👥 **إدارة العملاء** - إدارة بيانات العملاء والديون
- 📦 **إدارة المنتجات** - إدارة المخزون والفئات
- 📄 **إدارة الفواتير** - عرض وطباعة الفواتير
- 📈 **رسوم بيانية متقدمة** - تصور البيانات بطرق مختلفة
- 🔐 **مصادقة آمنة** - تسجيل دخول عبر Firebase
- 🌙 **واجهة عصرية** - تصميم احترافي وسهل الاستخدام
- 📱 **متجاوب** - يعمل على جميع الأجهزة

## 🚀 البدء السريع

### المتطلبات
- Node.js 16+
- npm أو pnpm

### التثبيت

```bash
# استنساخ المشروع
git clone <repository-url>
cd barq-dashboard

# تثبيت المكتبات
pnpm install

# إنشاء ملف .env
cp .env.example .env
```

### التطوير

```bash
# تشغيل خادم التطوير
pnpm dev

# سيفتح المتصفح على http://localhost:5173
```

### البناء للإنتاج

```bash
# بناء المشروع
pnpm build

# معاينة الإنتاج
pnpm preview
```
